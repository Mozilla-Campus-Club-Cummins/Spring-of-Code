How to use
Clone or download the project files to your local machine.
Open index.html file in a web browser.
Start adding tasks to the list.
To mark a task as completed, simply click on the task item.
To remove a task from the list, hover over the task item and click the "x" icon on the right side.

Files Included
This project includes the following files:
index.html - The main HTML file containing the task list.
style.css - The CSS file that styles the task list.
script.js - The JavaScript file that handles the functionality of the task list.

Features
Add tasks to the list
Mark tasks as completed
Remove tasks from the list
Save tasks to local storage
Retrieve saved tasks from local storage
